Project Overview

User Enters Website

The user visits the platform, creates an account or logs in securely, and accesses their personal dashboard.

User Creates Habits

The user defines daily goals such as exercise, reading, study time, or meditation and sets tracking preferences.

System Shows Today’s Habits

The system automatically displays all habits scheduled for the current day in an organized list.

User Marks Habits

The user updates habit status by marking tasks as completed, skipped, or pending for the day.

System Calculates Performance

The system analyzes completion rates, calculates consistency score, and evaluates daily performance automatically.

Daily Report Generated

A detailed report is generated showing progress summary, achievements, missed habits, and improvement suggestions.

User Views Progress History

The user reviews long-term performance trends, progress charts, and habit history to track improvement over time.
